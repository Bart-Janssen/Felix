Specialisatie

In het begin van dit semester heb ik moeten kiezen tussen drie leer richtingen (of een mix) waar ik mijzelf verder in wil specialiseren. Deze drie richtingen zijn:

  • Security engineering;
  • Blue teaming;
  • Red teaming.

Ik heb de security engineering richting gekozen omdat ik programmeren leuk vind. Vooral hoe ik applicaties beter en veiliger kan maken. Blue- en red teaming lijkt mij minder leuk om mijzelf daarin verder te specialiseren. Ik vind het ook leuk om te onderzoeken hoe het kan dat bepaalde security zo krachtig kan zijn, hierbij zit bijvoorbeeld het toepassen van encryptie, maar ook het onderzoeken hoe deze encryptie inhoudelijk werkt. Door de mix van het programmeren en het onderzoeken en toepassen van security leek dit voor mij de best passende richting.


Ik wil na deze opleiding misschien nog de master opleiding doen in de richting computer science, daar kan ik mijzelf nog verder specialiseren. Na die opleiding wil ik gaan werken, waar weet ik nog niet, maar waarschijnlijk wel waar ik veel te maken heb met security maar ook wat afwisseling heb met programmeren.

Hoe ik dit wil aanpakken is te lezen in het leerplan.

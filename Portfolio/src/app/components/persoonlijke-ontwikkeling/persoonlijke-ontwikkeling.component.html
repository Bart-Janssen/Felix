Personal Development

Ik heb vrij snel in het begin van dit semester mijn leerplan opgezet. In dit leerplan heb ik beschreven in welke onderdelen ik beter wil worden. De soft skils zoals teamwork, reflecteren en communiceren zijn een onderdeel hiervan.

Ik heb binnen het R&D project veel samen gewerkt met de groepsgenoten. Ik ben nog geen enkele keer te laat gekomen of helemaal niet aanwezig geweest. Ik heb veel samen gewerkt binnen de groep. Dit bestaat uit iedere dag een standup doen en reflecteren wat je eerder hebt gedaan, hoe dat ging en wat vandaag op de planning staat. Ook 'Pair programming' hebben we toegepast. Zo heb ik vaak met Luuk naar problemen gezocht en samen opgelost. Een voorbeeld hiervan is het probleemzoeken en tegelijk een update programmeren tot we het probleem gevonden hadden. Dit gebeurde vaan vanuit thuiswerken in verband met Corona.

Ik heb ook feedback gegeven op code of documenten van andere en visa versa. We hadden als afspraak dat als iemand een stuk code of document gemaakt had, dat dit bekeken werdt door iemand anders en hier feedback op gegeven werdt. Dit hebben we gedaan om de code en documenten beter resultaat te geven en zodat we ook leren om feedback te geven en te verwerken.

De leerdoelen die ik opgesteld heb aan het begin van het semester zijn niet verranderd. Veel van de leerdoelen zijn van de Canvas cursus, sommige leerdoelen heb ik zelf toegevoegd omdat ik hier graag meer van wilde leren. Ik heb er geen toegevoegd, weg gehaald of mijn planning aangepast. Hieruit blijkt dat ik de planning in het begin goed gemaakt had.
